# eldery-chatbot

노인을 위한 말벗 챗봇

### environmnet
python >= 3.5
flask == 1.0.2

### usage

1. 필요한 패키지 설치
	``pip3 install --user -r requirements.txt``

2. 웹 실행하기
	- localhost : ``python app.py``
	- [demo](http://hyunkyung12.pythonanywhere.com)
